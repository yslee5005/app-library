# IDEA.md — Abba (기도/QT AI 동반자)

> Created: 2026-04-06
> Status: VALIDATED

## YC 4P 검증

### Persona
- 전세계 기독교인 (23억명), 특히 50~70대
- 매일 기도/QT 하고 싶지만 습관이 안 잡히는 사람
- 기도 후 피드백이나 기록이 없어서 성장 실감 못 하는 사람
- 스마트폰은 쓰지만 복잡한 앱은 어려운 시니어

### Problem (Hair on fire?)
- 기도하면 끝 — 기록 없음, 피드백 없음, 성장 실감 없음
- QT 교재는 비싸고, 저작권 문제로 앱에서 제공 어려움
- 기존 앱(Glorify, Hallow)은 "콘텐츠 소비" 중심 — 내 기도에 대한 응답이 아님
- 히브리어/헬라어 원어 해석을 쉽게 접할 수 없음
→ Pain level: 8/10 (매일 반복, 영적 성장 직결)

### Promise
"기도하면 AI가 성경으로 응답하고, 당신을 위한 기도를 들려줍니다"

### Product (MVP)
1. 메인: 큰 녹음 버튼 하나 → 기도/QT 시작
2. 음성 녹음 (STT 온디바이스, 무료) + 텍스트 입력 가능
3. AI 대시보드:
   - 관련 성경 문구
   - 관련 성경 이야기
   - 간증 자동 기록
   - AI 조언 (유료)
   - 히브리어/헬라어 원어 해석 (유료)
   - AI 기도문 생성 + TTS 읽어주기 (유료, 목소리 선택)
4. QT 모드: AI가 매일 5개 성경 본문 생성 (cronjob, 저작권 free)
5. 기도 달력 + 스트릭
6. 커뮤니티: 익명/실명, 간증 공유, 댓글, 좋아요, 리플라이, 저장

### 핵심 차별점 (vs Glorify $6.99)
- Glorify = "콘텐츠를 소비하는" 앱
- **Abba = "내 기도를 AI가 응답하는" 앱**
- 음성 입력 (Glorify에 없음)
- AI 개인 맞춤 분석 (Glorify에 없음)
- AI 기도문 TTS (Glorify에 없음)
- 원어 해석 (Glorify에 없음)
- 기도 스트릭/달력 (Glorify에 없음)

| 기도/QT | 음성+텍스트, 1회/일 | 무제한 |

| 원어 해석 | ❌ | ✅ 히브리어/헬라어 |

연간: 프로모션 $29.99 → 정가 $49.99

### AI 비용 분석 (세션당)
| 구성 | 비용 |
|---|---|
| STT | $0 (온디바이스 Apple/Google) |
| LLM x3 (GPT-4o-mini) | $0.004 |
| TTS (OpenAI) | $0.008 |
| **합계** | **$0.012/세션** |

유료 유저: 75세션/월 = $0.90 비용, $4.89 수익(수수료 후) = **$3.99 마진**
손익분기: 유료 전환율 ~1%면 충분

### 기술 스택
| 기능 | 기술 |
|---|---|
| 프레임워크 | Flutter (iOS + Android) |
| 음성→텍스트 | 온디바이스 STT (speech_to_text 패키지) |
| AI 분석/기도문 | GPT-4o-mini API |
| 기도문→음성 | OpenAI TTS (목소리 선택) |
| 원어 해석 | 히브리어/헬라어 DB + AI 재창조 |
| QT 본문 생성 | AI cronjob (매일 5개) |
| 백엔드 | Supabase (app_id: 'abba') |
| 에러 로깅 | Sentry |
| 구독 결제 | RevenueCat |
| 다국어 | 영어 기본, 한국어, 확장 가능 |

### UX 원칙 (최우선)
- **60대 이상이 바로 보고 바로 하기**
- 큰 글씨 (최소 18pt)
- 큰 버튼 (최소 56dp)
- 메인 화면에 버튼 1개 (녹음 시작)
- 단순한 플로우 (3탭 이하)
- 높은 대비 (가독성)
- 최소한의 텍스트, 아이콘 중심

### 첫 100명 유저
- 한국/미국 교회 50~70대 성도
- 기독교 시니어 커뮤니티 (카카오톡 그룹, 교회 소그룹)
- "매일 기도하면 AI가 성경으로 응답" → 교회 소그룹에서 입소문

### Tar Pit 체크
