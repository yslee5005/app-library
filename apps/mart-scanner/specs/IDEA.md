## YC 4P 검증

### Persona
- 미국 거주 한국인 (약 250만명)
- 유학생, 이민자, 주재원, 워홀
- 영어 성분표 읽기 어렵거나 귀찮은 사람
- 건강에 관심 있는 사람, 알레르기 있는 사람

### Problem (Hair on fire?)
- 마트에서 제품 성분표를 영어로 읽어야 함 → 시간 소모
- 낯선 첨가물이 안전한지 모름 → 불안
- 알레르기 성분 확인을 매번 일일이 해야 함 → 놓칠 위험
- 결국 아는 제품만 계속 삼 → 선택 제한
→ Pain level: 8/10 (매주 반복, 건강 직결)

### Promise
"바코드 스캔 한 번으로 3초 안에 한국어 건강 분석"

### Product (MVP)
1. 바코드 스캔 → 제품 DB 조회
2. 한국어 번역 (제품명, 성분, 영양정보)
3. 건강 점수 (A~E 등급)
4. 알레르기 경고 (내 알레르기 프로필 기반)
5. 제품 비교 (2~4개)

### 기존 대안
- Yuka: 건강 점수 O, 한국어 X
- Open Food Facts: 데이터 O, UX 구림, 한국어 X
- 구글 번역: 성분 하나씩 번역, 건강 판단 X

→ **한국어 + 건강 점수 + 알레르기 = 경쟁자 없음**

### 첫 100명 유저
- 미국 한인 커뮤니티 (카카오톡 그룹, 한인 교회, 유학생 커뮤니티)
- 레딧 r/korean, r/KoreanAmerican
- "미국 마트 장보기" 네이버 카페

### Tar Pit 체크
- [ ] App Store에 비슷한 앱이 수십 개? → ❌ 한국어는 없음
- [ ] 돈을 낼 사람이 불명확? → ❌ 매주 장보는 사람은 확실히 필요
- [ ] 기술적으로만 재미있고 사용자 문제가 약? → ❌ 실제 pain 매우 강함
