# BabyLetter Stitch Design Brief — 전체 화면 상세 정의

> **디자인 방향**: 밋밋한 UI → Lovely + Gamification + 따뜻한 일러스트
> **핵심 톤**: 편지를 받는 감성, 아기와 함께 성장하는 게임, 따뜻한 수채화 느낌
> **사용 에셋**: Zone A/B/C 태아 성장 영상 프레임 (47개 stage)

---

## Gamification 시스템 (전역)

### 1. 성장 레벨 시스템
- **씨앗 Lv.1** → **새싹 Lv.5** → **꽃봉오리 Lv.10** → **꽃 Lv.20** → **열매 Lv.30**
- 기록할 때마다 XP +10, 편지 읽으면 XP +5, EPDS 완료 XP +30
- 레벨업 시 축하 애니메이션 + 새 배경/프레임 잠금해제

### 2. 연속 기록 스트릭
- 불꽃 아이콘 + "연속 7일째 기록 중!"
- 3일/7일/14일/30일/100일 마일스톤 보상 (배지)
- 놓쳐도 비판단적: "쉬어가는 날도 괜찮아요"

### 3. 아기 캐릭터 진화
- 임신: 콩 → 포도 → 레몬 → 아보카도 → 수박 (크기 비유 캐릭터)
- 출산후: 신생아 → 뒤집기아기 → 앉는아기 → 기는아기 → 걷는아기
- 홈화면 중앙에 항상 현재 단계 캐릭터 표시

### 4. 배지 컬렉션
- 🍼 수유마스터 (수유 100회)
- 😴 잠의요정 (수면기록 30일)
- 💩 기저귀전사 (기저귀 200회)
- 💌 편지수집가 (편지 28개 모두 읽기)
- 🧠 뇌과학자 (S&R 가이드 5회 완료)
- 👨 슈퍼아빠 (아빠미션 20개 완료)
- 💛 삼각형케어 (EPDS 엄마+아빠 둘다 4주 연속)
- 📊 패턴마스터 (7일 연속 전체기록)

### 5. 일일 보상
- 매일 첫 기록 시: "오늘의 선물 상자" 열림 → 랜덤 격려 메시지/팁/귀여운 스티커

## Zone A: 온보딩 (7 screens)

---

### A1. 스플래시 화면
**현재**: 단순 텍스트 + 그라데이션
**Stitch 디자인**:
- 중앙에 편지봉투 일러스트 (수채화 스타일, 코랄+앰버 톤)
- 봉투가 천천히 열리며 하트가 날아오르는 Lottie 애니메이션
- 배경: Cream → Amber 부드러운 그라데이션
- 하단에 "아기의 편지" 로고 (Plus Jakarta Sans, 손글씨 느낌의 밑줄)
- 작은 별 파티클이 천천히 떠다님
- **소요 시간**: 2초 자동 전환

---

### A2. 웰컴 — 모드 선택
**현재**: 단순 두 버튼
**Stitch 디자인**:
- 상단 60%: 엄마+아기 수채화 일러스트 (따뜻한 톤)
- 중앙 텍스트: "안녕하세요, 엄마! 아기가 편지를 보내기 시작했어요" (필기체 느낌)
- 두 개의 큰 카드 버튼:
  - 🤰 **"아직 배 속에 있어요"** — 임신 중 (핑크 그라데이션 카드 + 작은 배아 일러스트)
  - 👶 **"이미 만났어요!"** — 출산 후 (앰버 그라데이션 카드 + 아기 얼굴 일러스트)
- 카드 hover 시 살짝 커지는 scale 애니메이션
- 하단: 부드러운 물결 패턴 데코레이션

---

### A3. 정보입력 — 임신 중 (3단계)
**현재**: 기본 폼
**Stitch 디자인**:
- **Step 1 — 출산예정일**:
  - 상단에 아기 캐릭터 + 말풍선 "엄마, 언제 만나?"
  - 캘린더 위젯 (coral 포인트, 선택 날짜에 하트)
  - 주차 자동 계산 표시: "지금 24주 3일이에요!"
- **Step 2 — 태명**:
  - 아기 캐릭터 + 말풍선 "내 이름이 뭐야?"
  - 텍스트 입력 + 입력하면 캐릭터가 손 흔들기 애니메이션
  - 건너뛰기 옵션 (비판단적)
- **Step 3 — 파트너 초대**:
  - 삼각형 일러스트 (아기-엄마-아빠 자리에 ?)
  - "함께하면 더 좋아요" 메시지
  - 카카오톡/링크 공유 버튼
  - "나중에 할게요" 텍스트 버튼
- **공통**: 상단 진행바 (3개 도트, 현재 도트에 pulse 효과)

---

### A4. 정보입력 — 출산 후 (3단계)
**Stitch 디자인**:
- **Step 1 — 생년월일**:
  - 아기 캐릭터 (신생아) + 말풍선 "나 태어났어!"
  - 날짜 선택 → "D+45 우리 아기!" 자동 표시
- **Step 2 — 이름/성별**:
  - 성별 선택: 파란 카드(남아) / 핑크 카드(여아) / 보라 카드(비공개)
  - 이름 입력 → 아기 캐릭터 위에 이름 뱃지 표시
- **Step 3 — 수유방식**:
  - 3개 일러스트 카드: 🤱 모유 / 🍼 분유 / 🔄 혼합
  - 비판단적 메시지: "어떤 방법이든 아기는 잘 자라요"

---

### A5. 크로노타입 퀴즈 (G1)
**현재**: 기본 라디오 버튼
**Stitch 디자인**:
- 배경: 밤하늘 그라데이션 (남색 → 보라)
- 달/별 일러스트 장식
- 5문항 카드 스와이프 (카드 둥근 모서리 + 그림자)
- 각 선택지에 귀여운 아이콘: 🌅 아침형 / ☀️ 낮형 / 🌙 저녁형
- 결과 화면: 올빼미/종달새/비둘기 캐릭터 일러스트 + 성격 설명
- 하단: "이 정보가 아기 수면 스케줄 예측에 도움이 돼요!" 메시지

---

### A6. 기질 퀴즈 (G2)
**Stitch 디자인**:
- 배경: 부드러운 파스텔 그라데이션
- 8문항, 각 문항에 아기 행동 일러스트
- 선택지: 슬라이더 형태 (왼쪽 "전혀" ↔ 오른쪽 "매우")
- 결과: 4가지 기질 캐릭터
  - 🌈 순한아기 (40%) — 무지개 배경
  - 🔥 활발한아기 (10%) — 오렌지 배경
  - 🦉 신중한아기 (15%) — 파란 배경
  - 🎨 혼합아기 (35%) — 멀티컬러 배경
- 5축 레이더 차트를 꽃잎 모양으로 시각화

---

### A7. 수면환경 설정 (G3)
**Stitch 디자인**:
- 아기 방 일러스트 배경 (따뜻한 조명)
- 3개 대형 카드 (높이 120px):
  - 🛏️ 독립 수면 — 아기 침대 일러스트
  - 🤱 함께 자기 — 부모+아기 일러스트
  - 🔄 혼합 — 두 장면 split 일러스트
- 선택 시: 카드 테두리 coral 글로우 + 체크마크 뱃지
- 비판단적 푸터: "정답은 없어요. 우리 가족에게 맞는 방법이 최고예요"

---

## Zone B: 홈탭 (11 screens)

---

### B1. 홈탭 — 임신 중
**현재**: 밋밋한 카드 나열
**Stitch 디자인**:
- **상단 헤더**:
  - 좌: 아기 캐릭터 (현재 주차 크기비유, 예: 24주=옥수수) + 레벨 배지
  - 중: "24주 3일" (큰 숫자) + "옥수수만해졌어요!" 서브텍스트
  - 우: 🔥 스트릭 카운터 + 알림 벨
- **태아 영상 카드** (Hero):
  - 둥근 사각형 (16px), Zone B 태아 이미지 배경
  - 하단에 반투명 오버레이: "Week 24 — 약 30cm, 600g"
  - 재생 버튼 (원형, pulse 애니메이션)
  - 좌우 스와이프 힌트 화살표 (주차 이동)
- **오늘의 편지 카드**:
  - 봉투 모양 카드, 살짝 열린 상태
  - "읽지 않은 편지 1통" 뱃지 (빨간 도트)
  - 탭하면 봉투 열리는 애니메이션 → B3
- **XP 진행 바** (인라인):
  - 현재 레벨 아이콘 + 프로그레스 바 + 다음 레벨 아이콘
  - "32 XP 더 모으면 새싹 레벨!" 텍스트
- **오늘의 미션 카드** (Gamification):
  - 체크리스트 형태: ☐ 편지 읽기 / ☐ 태담하기 / ☐ 발달팁 확인
  - 완료 시 스탬프 도장 효과 + XP 획득 토스트
- **발달 팁 카드**:
  - 일러스트 + 한줄 팁 + "자세히 보기" 화살표
  - 좌우 스와이프 (오늘/어제/그제)
- **Serve & Return 카드**:
  - 🧠 뇌 아이콘 + "오늘의 실천" 텍스트
  - 귀여운 말풍선 형태의 실천 내용
- **4th Trimester 카드** (30주+):
  - ⚡ "D-70 출산 카운트다운" 배지
  - 미니 체크리스트 프리뷰 + 진행도 링

---

### B2. 홈탭 — 출산 후
**Stitch 디자인**:
- **상단 헤더**:
  - 아기 캐릭터 (현재 발달 단계) + 레벨 배지 + 이름
  - "D+45 (1개월 15일)" 큰 숫자
  - 🔥 스트릭 + 알림 벨
- **오늘의 성장 카드** (Hero):
  - 아기 캐릭터 중앙 + 기질유형 뱃지 ("활발한 아기 🔥")
  - 하단 3개 미니 지표 (수면/수유/기저귀) 아이콘 + 숫자
  - 원형 프로그레스: "정상 범위" 게이지
  - 배경: 부드러운 그라데이션 (기질유형별 다른 색)
- **오늘의 편지 카드**: (B1과 동일 패턴)
- **XP + 스트릭 인라인 바**: (B1과 동일)
- **오늘의 미션 카드**:
  - ☐ 기록하기 / ☐ 편지 읽기 / ☐ 아기와 5분 놀기
- **Serve & Return 카드**:
  - 실천 카운터: "이번 주 3/5회 달성!"
  - 프로그레스 바 + 주간 목표 배지
- **어제의 회고 카드**:
  - 미니 요약 (수면/수유/기저귀 아이콘 + 숫자)
  - "지금은 아기와 함께하는 시간이에요 💛" Technoference 메시지
- **퀵기록 FAB**: 하단 우측 플로팅 버튼 (coral, + 아이콘)

---

### B3. 아기편지 상세
**현재**: 기본 텍스트 카드
**Stitch 디자인**:
- 전체 화면 편지지 배경 (크림색 + 미세한 종이 텍스처)
- 상단: 봉투 열리는 애니메이션 (첫 진입 시)
- 편지 본문:
  - 필기체 스타일 폰트 (or 둥근 폰트)
  - 아기 캐릭터 프로필 원 + "24주의 콩이가"
  - 본문 텍스트 (한글자씩 타이핑 효과)
  - 주요 키워드에 하이라이트 (연한 coral 배경)
- 과학적 근거 섹션:
  - 접히는 아코디언: "📚 과학적 근거" 탭
  - 논문 출처 + 간단 설명
- 하단:
  - ❤️ 하트 버튼 (좋아요/보관) + 공유 버튼
  - "다음 편지는 내일 아침에 도착해요" 메시지
  - XP +5 획득 토스트 (첫 읽기 시)

---

### B4. 태아영상 풀스크린
**Stitch 디자인**:
- 전체 검정 배경
- 중앙: Zone A/B/C 실제 태아 이미지/영상 (1:1)
- 하단 반투명 오버레이:
  - 주차 + 크기 + 발달 하이라이트 텍스트
  - 크기 비유 일러스트 (예: "포도알만해요 🍇")
- 좌우 화살표 (이전/다음 주차)
- 하단 미니 타임라인 바 (현재 위치 도트)
- 상단: ✕ 닫기 버튼 + 공유 버튼

---

### B5. 편지 히스토리 (편지 모음)
**현재**: 기본 리스트
**Stitch 디자인**:
- 상단: "💌 아기의 편지 모음" + 총 N통 배지
- 필터 칩: 전체 / 임신 중 / 출산 후 / ❤️ 보관함
- **컬렉션 그리드** (2열):
  - 각 편지 = 미니 봉투 카드 (봉투 일러스트)
  - 읽은 편지: 열린 봉투 + 프리뷰 텍스트
  - 안 읽은 편지: 닫힌 봉투 + "New" 뱃지 + 빛나는 효과
  - 잠긴 편지: 회색 봉투 + 자물쇠 + "28주에 도착 예정"
- **수집 진행도**: "28통 중 16통 수집! (57%)" 프로그레스 바
- 전부 수집 시: 🏆 "편지수집가" 배지 잠금해제 축하

---

### B6. Serve & Return 가이드
**현재**: 기본 스텝 카드
**Stitch 디자인**:
- 상단: 🧠 뇌+하트 일러스트 + "Serve & Return이란?"
- Harvard 5단계를 **게임 스텝 맵**으로 표현:
  - 구불구불한 경로 위에 5개 체크포인트
  - 완료한 단계: 컬러 + 체크 / 미완료: 회색 + 잠금
  - 각 단계 탭 → 카드 플립 애니메이션으로 상세 설명
- **오늘의 실천 챌린지**:
  - "2초 안에 응답하기" 미니게임 컨셉
  - 타이머 + 성공 시 별 이펙트
  - 주간 성공률 차트 (미니 바)
- **참고 자료**: Still Face 실험 영상 링크 카드

---

### B7. 4th Trimester 가이드
**Stitch 디자인**:
- 상단: 자궁 → 세상 비주얼 (split 일러스트)
  - 왼쪽: 따뜻한 자궁 일러스트 (어둡고 따뜻)
  - 오른쪽: 밝고 시끄러운 세상 (밝고 차가움)
  - 화살표 + "이 변화를 상상해보세요"
- **4개 기간 탭** (탭 바):
  - 0-2주 🐣 / 2-6주 🌱 / 6-12주 🌿 / 3-6개월 🌻
  - 각 탭에 해당 기간 일러스트 아이콘
- **체크리스트**: 기간별 준비물/실천 항목
  - 완료 시 스탬프 효과 + XP
- **D-Day 카운트다운 배지**: "출산까지 D-42"
- 하단: 아기 시점 편지 말풍선

---

### B8. 위험신호 상세
**Stitch 디자인**:
- ⚠️ **풀스크린 경고 모드**:
  - 배경: 연한 빨강 그라데이션 (공포 주지 않되 주목)
  - 상단: 큰 ⚠️ 아이콘 + "위험신호 감지"
