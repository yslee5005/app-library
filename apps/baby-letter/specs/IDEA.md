## Persona
**임신 20주~출산 후 12개월 초보 부모 (25-38세)**
- 첫 아이 임신/출산한 부부
- 아기 발달에 대한 정보가 넘쳐서 오히려 불안한 사람
- "내 아기가 정상인지" 걱정하지만 의사에게 뭘 물어야 할지 모르는 사람
- 아빠는 "나는 뭘 해야 하지?" 막막한 사람

## Problem (Hair on Fire)
1. **정보 과잉 + 판단의 공포**: 기존 앱들이 "평균보다 적습니다" 같은 판단을 내려서 불안 유발
2. **아빠 소외**: 모든 육아앱이 엄마 중심 → 아빠는 관찰자 역할만
3. **산후우울증 사각지대**: 엄마 80% Baby Blues, 아빠 8-15% 우울증 — 조기 발견 앱 없음
4. **Technoference**: 아이러니하게 육아 추적 앱이 부모-아기 상호작용을 방해
5. **Cold Start 문제**: 출산 후 처음 앱 깔면 아기 정보가 0 → 개인화 불가

## Promise
**"아기가 보내는 편지로 시작하는, 판단하지 않는 육아 앱"**
- 기록은 하되 판단하지 않음 (저울처럼 숫자만 보여줌)
- 아기 시점 1인칭 편지로 발달 과학을 감성적으로 전달
- 엄마+아빠+아기 삼각형 케어 모델
- 임신 중부터 시작해서 출산 시 Warm Start

## Product (MVP)
### 핵심 기능 5개
1. **아기의 편지**: 주차별 발달 과학 기반 1인칭 메시지 (임신 8주~출산 후 12개월)
2. **비판단적 기록**: 수유/수면/기저귀 기록 + 판단 없는 패턴 시각화
3. **4-Layer 프로파일링**: 연령→수유방식→기질→개인베이스라인 자동 분류
4. **삼각형 케어 (우리탭)**: EPDS 양육자 정서 체크 + 파트너 메시지 + 아빠 미션
5. **절대 위험만 알림**: 회백색 대변, 혈변 등 연 2-3회만 울리는 알림 (신뢰 구축)

### MVP에서 제외
- 커뮤니티/SNS 기능
- AI 챗봇 상담
- 쇼핑/광고
- 실시간 모니터링 (의도적 제외 — Technoference 방지)

## 첫 100명 유저
1. **산부인과/산후조리원 제휴**: 임신 20주 검진 시 앱 소개 전단
2. **맘카페 (네이버 맘스홀릭)**: "아기가 보내는 편지" 컨셉으로 바이럴
3. **아빠 커뮤니티**: 아빠 미션 기능으로 "아빠도 쓰는 앱" 포지셔닝
4. **Instagram**: 주차별 태아 영상 + 아기편지 콘텐츠 공유
5. **출산 준비 클래스**: 4th Trimester 교육 콘텐츠로 신뢰 구축
