# 아기의 편지 (BabyLetter) — DESIGN.md

> **관련 문서**: [ROADMAP.md](./ROADMAP.md) | [IDEA.md](./IDEA.md) | [REQUIREMENTS.md](./REQUIREMENTS.md) | [SCREEN_CATALOG.md](./SCREEN_CATALOG.md)

## 디자인 원칙
1. **따뜻함 (Warm)**: 의료 앱이 아닌 "편지를 받는" 감성
2. **비판단적 (Non-judgmental)**: 숫자는 보여주되 "좋다/나쁘다" 텍스트 없음
3. **아기 시점 (Baby's Voice)**: 1인칭 편지, 아기 코멘트가 곳곳에
4. **삼각형 (Triangle)**: 항상 엄마+아빠+아기 세 축이 보임
5. **Technoference 방지**: "앱을 내려놓으세요" — 회고 형태, 5초 확인
6. **점진적 복잡성**: 처음엔 단순 → 데이터 쌓이며 인사이트 증가

## 카피라이팅 원칙

### 공식: [감정 Hook] + [과학 사실] + [행동 유도]
- ❌ "24주입니다. 청각이 발달합니다." (단순 나열)
- ✅ "콩이가 엄마 목소리를 구별하기 시작했어요 🎵 오늘 3분만 노래 불러줄래요?"

### 규칙
1. **모든 숫자에 맥락 문장 동반** — "14.2시간" → "콩이랑 비슷한 아기들 평균 14시간, 콩이도 14.2시간! 잘 자고 있어요"
2. **아기 이름(태명) 항상 포함** — 개인화된 경험
3. **행동 유도형 CTA** — "~해볼래요?", "~해보세요"
4. **비판단적이지만 격려적** — "잘하고 있어요", "괜찮아요"
5. **매일 다른 문구** — 반복 = 이탈
6. **기록 완료 시 아기 코멘트 피드백** — "콩이가 배불러해요! 💛"

### 화면별 카피 톤
| 화면 | 톤 | 예시 |
|------|-----|------|
| 홈 아침인사 | 따뜻한 환기 | "좋은 아침! 콩이는 오늘 하품 연습 중이에요 🥱" |
| 편지 | 아기 1인칭 감정 | "엄마, 나 오늘 딸꾹질 했어! 폐 연습하는 거래" |

## 비교군 시각화 원칙

### 게이지 바 (판단 없는 위치 표시)
```
  😴 수면
  ◀──────[===콩이===]──────▶
     12h    14.2h      16h
            ↑
  활발한 아기 평균 14.0h
  "잘 자고 있어요! 계속 이대로 💛"
```

- 중립 톤 사용 (coral 도트 + 회색 바)
- "많다/적다" 대신 "비슷한 아기들은 평균 Xh"
- 정상 범위는 넓은 밴드로 표시 (좁은 기준선 X)
- 절대 빨강/초록 컬러 코딩 하지 않음 (모두 중립 톤)

## 가족 소통 시스템

### 1. 매일 감정 체크인
- 4단계 이모지: 😊 좋아 / 😐 그냥 / 😢 지침 / 😤 힘들어
- 1탭 공유 → 파트너에게 자동 전달
- 상대방 상태를 홈에 미니 표시
- 무거운 EPDS 대신 매일 가벼운 감정 공유

### 2. 배려 넛지 (자동)
- 파트너 😢 → "수고했어라고 한마디 해볼까요?"
- 3일 연속 😐 이하 → "30분 아기 맡아볼까요?"
- 비난 없이, 제안 형태로만

### 3. 양방향 가족 미션
- 기존 아빠 미션(일방적) → 엄마+아빠+함께 미션(양방향)
- 주간 3+3+2 = 8개 미션
- 완료 시 가족 XP 획득
- 예시:
  - 👩 엄마: "아빠에게 감사 표현하기", "30분 혼자만의 시간 갖기"
  - 👨 아빠: "아기 목욕시키기", "엄마 30분 자유시간 주기"
  - 👨‍👩‍👧 함께: "아기 사진 찍기", "10분 대화 (아기 말고)"

### 4. 감사 저널 (하루 마무리)
- "오늘 가장 좋았던 순간은?" 텍스트 입력
- 사진 첨부 가능
- 아기 코멘트 자동: "엄마, 오늘도 고마워! 내일도 같이 놀자!"
- Retention 효과: 매일 저녁 앱 오픈 유도

## Retention 전략

### 매일 오픈 유도 (3-touch)
| 시간 | 트리거 | 내용 |
|------|--------|------|
| 아침 | 아기 편지 푸시 | 감정 Hook: "콩이가 새 편지를 보냈어요" |
| 낮 | 기록 리마인더 | 가벼운: "마지막 수유가 3시간 전이에요" |
| 저녁 | 감사 저널 | "오늘 가장 좋았던 순간은?" |

### 주간 사이클
| 요일 | 이벤트 |
|------|--------|
| 월 | 새 가족 미션 공개 |
| 수 | EPDS 리마인더 (선택) |
| 금 | 주간 회고 생성 |
| 일 | 다음 주 마일스톤 예고 |

### Gamification
- 기록 = XP +10 / 편지 읽기 = XP +5 / EPDS = XP +30 / 마일스톤 = XP +50
- 레벨: 씨앗 → 새싹 → 꽃봉오리 → 꽃 → 열매
- 연속 기록 = 스트릭 불꽃 (3/7/14/30/100일 마일스톤)
- 배지 8종: 수유마스터, 잠의요정, 기저귀전사, 편지수집가, 뇌과학자, 슈퍼아빠, 삼각형케어, 패턴마스터

### Anti-Churn (이탈 방지)
| 조건 | 대응 |
|------|------|
| 3일 미접속 | "콩이가 기다리고 있어요 💛" 푸시 |
| 7일 미접속 | "돌아올 때까지 편지 모아둘게요" 푸시 |
| 기록 빈 날 | "쉬어가는 날도 괜찮아요" (비판단적) |
| 앱 5분 사용 | "아기에게 돌아가세요 💛" (Technoference 방지) |
