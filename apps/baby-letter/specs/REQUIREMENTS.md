## 앱 이름
아기의 편지 (BabyLetter)

## 한줄 설명
아기가 보내는 1인칭 편지로 발달 과학을 전달하고, 판단 없이 기록하며, 엄마+아빠+아기 삼각형 케어를 지원하는 앱

## Bundle ID
com.ystech.babyletter

## 핵심 기능

### 1. 온보딩 (Progressive Profiling)
- 임신 중 / 출산 후 분기
- 임신 중: 출산예정일 + 태명(선택) + 파트너초대(선택) → 3문항
- 출산 후: 생년월일 + 이름/성별 + 수유방식 → Layer 1+2 즉시 완성
- Anonymous-First: 로그인 없이 시작

### 2. 홈탭
#### 임신 중
- 태아 성장 영상 카드 (47개 영상, 1080x1080 루프)
- 아기의 편지 (주차별 1인칭 메시지)
- 오늘의 발달 팁 (과학 기반 + 출처)
- Serve & Return 실천 가이드
- 4th Trimester 준비 (30주+)

#### 출산 후
- 오늘의 성장 요약 (기질유형 뱃지 + 주요 지표)
- 아기의 편지 (일일 1인칭 메시지)
- Serve & Return 실천 가이드
- 어제의 회고 (Technoference 방지: 실시간 X, 회고 O)
- 위험신호 (조건부 — 해당 시에만)

### 3. 기록탭
#### 임신 중
- 태동 카운터 (Count-to-10 방식)
- 체중/혈압/기분/메모 일지
- 태동 패턴 시각화

#### 출산 후
- 수유 기록 (모유 좌/우 타이머 + 분유 ml + 혼합)
- 수면 기록 (시작/종료 + 낮잠/밤잠 + 수면환경 태그)
- 기저귀 기록 (소변/대변 + 색상 7색 팔레트 + 사진)
- 빠른 기록 퀵액션 바텀시트
- 패턴 시각화: 일간(24h 타임라인) + 주간(7일 비교 + 개인 베이스라인)
- 캘린더 뷰

### 4. 성장탭
#### 임신 중
- 수직 타임라인 (Zone A/B/C, 47개 노드)
- 노드 상세: 태아 영상 + 크기 비교 이미지 + 발달 키워드 + 아기 편지
- 주차별 검진 체크리스트

#### 출산 후
- 아기 프로필 카드 (4-Layer 프로파일링 결과)
- 같은유형 비교 (판단 없이 시각적으로만)
- 마일스톤 체크리스트 (CDC 기반 + "아직 안 해도 괜찮아요")
- 소아과 리포트 (2주간 자동 요약 + PDF 내보내기)
- 성장 차트 (키/몸무게 WHO 백분위)

### 5. 우리탭 (삼각형 케어)
- 삼각형 비주얼 (아기-엄마-아빠)
- EPDS 설문 (엄마/아빠 각각, 주간)
- EPDS 결과 3단계: 정상(초록)/주의(노랑)/위험(빨강)
- 위험 시 즉시 도움 연결 (핫라인 번호)
- 파트너 메시지 (감정 카드 보내기/받기)
- 아빠 미션 (주간 3개 미션 + 아기 시점 코멘트)
- 주간 회고

### 6. 설정
- 프로필 관리 (아기 정보 + 파트너 + 계정 연결)
- 알림 설정 (카테고리별 ON/OFF)
- 데이터 내보내기/삭제

### 7. 프로파일링 (Progressive)
- 크로노타입 퀴즈 (임신 20주, 5문항)
- 기질 퀴즈 (출산 후 Day 3-7, 8문항)
- 수면환경 설정 (첫 수면 기록 시)
- 자동 추론 (Layer 4: 7일+ 데이터 축적 후)

### 8. 알림 시스템
- 아기편지: 매일 아침 푸시
- 위험신호: 풀스크린 경고 (대변 회백색, 혈변 등)
- EPDS 리마인더: 주간
- 파트너 메시지: 즉시
- 마일스톤 축하: 달성 시 컨페티

## 앱 고유 기능
- 아기편지 콘텐츠 엔진 (주차별 매핑 + 1인칭 생성)
- 4-Layer 프로파일링 시스템
- 태아 성장 영상 47개 에셋
- EPDS 설문 + 점수 계산 + 3단계 분류
- 태동 카운터 (Count-to-10)
- 대변 색상 위험 감지
- 파트너 메시지 시스템
- 아빠 미션 시스템
- Serve & Return 가이드 콘텐츠
- 4th Trimester 가이드 콘텐츠
