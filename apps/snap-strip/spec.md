# snap-strip

## 한 줄 컨셉
앱으로 인생네컷(4컷 스트립 사진)을 찍고, 가까운 CVS·Walgreens 매장에서 인화된 사진을 픽업하는 서비스.

## 타겟 시장
미국 (Photo booth 문화 + 약국 체인 인쇄 인프라가 결합 가능한 환경)

## 타겟 사용자
- 10~20대 Z세대: 친구/커플/가족과 포토부스 감성 사진을 즐기는 층
- 실물 사진을 빠르게 손에 넣고 싶은 사용자 (배송 대기 없이 당일 픽업)

## 핵심 플로우
1. **촬영** — 앱에서 4컷 연속 촬영 (필터·프레임·스티커 적용)
2. **편집** — 스트립 레이아웃 자동 생성 (2x6 inch 규격)
3. **매장 선택** — 위치 기반으로 가까운 CVS / Walgreens 찾기
4. **결제** — 앱 내 결제 (Stripe 등)
5. **주문 전송** — 인쇄 파트너 API로 매장에 주문 전달
6. **픽업** — 인쇄 완료 푸시 알림 → 매장 방문 → QR/주문번호로 수령

## 차별화 포인트
- 한국식 "인생네컷" 감성 프레임·필터·폰트를 미국 시장에 이식
- 즉석 프린트가 아닌 **매장 픽업 방식**으로 기기 투자·부스 운영 비용 제로
- 배송 대기 없이 **당일 1시간 내 수령** (Walgreens/CVS 1-hour print 활용)

## 인쇄 파트너 전략
- **MVP**: Printicular API (Walgreens·CVS·Walmart 통합, 개발자 등록 기반)
- **Phase 2**: Walgreens Photo Prints API 직접 연동 (수수료 절감)
- **Phase 3**: CVS 직접 B2B 파트너십

## 기술 방향 (초안)
- Frontend: Flutter (monorepo 일관성 유지)
- Backend: 이미지 저장/주문 관리 + 결제 중계 서버
- 외부 API: Printicular(인쇄), Stripe(결제), 위치(Google Maps)

## 주요 제약
- 2x6 inch 스트립 규격이 인쇄 파트너 API에서 지원되는지 확인 필요
- 미국 리테일 인쇄 표준은 4x6 / 5x7 중심 → 4x6 내 4컷 배치로 대체 가능성 검토
- 매장 픽업 인증 방식 (QR, 주문번호, 본인확인) 파트너별 정책 확인 필요
