# BlackLabelled 웹사이트 — IDEA.md

> YC 4P 검증

## Persona (누가 쓰나?)
- **1차**: 30~50대 성남/분당 거주 아파트 소유자. 인테리어 리모델링 계획 중.
- **2차**: 신혼부부, 전셋집 → 자가 이사 예정. "제대로 된" 인테리어 원함.
- **3차**: 상업공간 운영자 (카페, 병원, 헬스장). 공간 브랜딩 필요.

## Problem (무슨 문제?)
- 인테리어 업체 웹사이트가 전부 비슷함 (사진 나열)
- "이 업체가 진짜 잘하는지" 판단할 근거가 부족
- 도면만 보고 결과물을 상상하기 어려움
- 427개 프로젝트인데 현재 사이트가 느리고 탐색이 불편

## Promise (뭘 해결?)
"설계에서 현실로 — 당신의 공간이 어떻게 변하는지 직접 보여드립니다."

## Product (MVP로 뭘 만들면?)
- 다크 럭셔리 웹사이트 + 3D/인터랙티브 갤러리
- 도면→실제 사진 Before/After 슬라이더 (69쌍)
- 지도 기반 프로젝트 탐색 (Mapbox Dark)
- 3D Depth Gallery (스크롤 반응형)
- 모바일 최적화
