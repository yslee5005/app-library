# Claude Code Tools Documentation

## 개요
Claude Code는 Claude에게 로컬 머신과 상호작용할 수 있는 빌트인 도구 세트를 제공합니다. 이 도구들은 권한 모드와 규칙에 따라 제어됩니다.

## File Tools (파일 도구)

### Read
- **기능**: 로컬 파일에서 최대 2,000줄을 검색
- **매개변수**: offset/limit 지원
- **지원 형식**: 이미지(PNG, JPG), PDF(최대 20페이지), Jupyter 노트북
- **권한**: 읽기 전용, default 모드에서 자동 승인

### Edit
- **기능**: 대상 문자열 교체를 수행
- **전제 조건**: 사전 파일 읽기 필수
- **매칭 방식**: 정확한 매칭으로 의도하지 않은 변경 방지
- **다중 매칭**: 검색 문자열이 여러 번 나타나면 실패, `replace_all: true` 설정 시 모두 교체

### Write
- **기능**: 새 파일 생성 또는 기존 파일 완전 덮어쓰기
- **용도**: 수정보다는 새 파일 생성에 권장
- **전제 조건**: 기존 파일의 경우 사전 읽기 필요

### Glob
- **기능**: 패턴 매칭 파일 검색
- **지원 구문**: `**/*.ts`, `src/**/*.test.js` 등
- **정렬**: 수정 시간 기준 정렬 결과 반환

## Shell Tool (셸 도구)

### Bash
- **기능**: 지속적 셸 명령어 실행
- **복합 연산자 지원**: `&&`, `||`, `|`
- **백그라운드 실행**: `run_in_background: true` 옵션
- **출력 관리**: 대용량 결과에 대한 출력 잘림 처리

## Search Tools (검색 도구)

### Grep
- **엔진**: Ripgrep 기반 정규식 검색
- **출력 모드 3가지**:
  - 파일만 표시 (files-only)
  - 컨텍스트 포함 내용 (content with context)
  - 매칭 횟수 (match counts)
- **멀티라인**: 멀티라인 패턴 지원

### LS
- **기능**: 구조화된 형식의 디렉토리 목록
- **용도**: 프로젝트 탐색

## Web Tools (웹 도구)

### WebFetch
- **기능**: URL 콘텐츠 가져오기
- **변환**: HTML을 마크다운으로 변환
- **캐싱**: 15분 캐시
- **보안**: HTTP를 HTTPS로 자동 업그레이드

### WebSearch
- **기능**: 웹 검색 수행, 제목 및 URL 포함 결과 반환
- **도메인 필터링**: 현재 US 전용

## Agent Tools (에이전트 도구)

### Task
- **기능**: 격리된 컨텍스트를 가진 서브 에이전트 생성
- **용도**: 다단계 검색 또는 병렬 워크스트림
- **특징**: 독립된 대화 루프 실행

### TodoWrite
- **기능**: 구조화된 작업 목록 관리
- **상태 추적**: pending, in_progress, completed

## MCP Tools

- MCP(Model Context Protocol) 서버에서 제공하는 추가 도구
- `mcp__` 접두사로 표시
- 권한 규칙으로 거부 가능

## 도구 가용성 결정 요인

도구 접근은 다음에 의해 결정됩니다:

1. **환경 변수**: 특정 환경에서만 활성화되는 도구
2. **권한 규칙**: allow/deny 목록으로 제어
3. **활성화/비활성화 설정**: 도구별 토글
4. **MCP 서버 연결 상태**: 서버 연결 여부에 따른 가용성
