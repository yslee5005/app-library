# Claude Code 마스터 코스

> AI 코딩 에이전트의 시대, 제대로 배우고 제대로 써먹자.

---

## 이 가이드를 만든 이유

2026년 초 기준, 개발자의 46%가 "가장 좋아하는 AI 코딩 도구"로 Claude Code를 꼽았다. Cursor(19%)와 GitHub Copilot(9%)을 크게 앞선다. 하지만 대부분의 개발자는 Claude Code의 잠재력 중 10%도 활용하지 못하고 있다.

"프롬프트 넣으면 코드 나오는 거 아냐?" -- 맞다. 하지만 그건 자동차를 1단 기어로만 모는 것과 같다. 에이전틱 루프가 어떻게 돌아가는지 이해하고, 메모리 시스템을 제대로 설정하고, Hook과 Skill로 워크플로우를 자동화하면, 같은 도구로 10배의 생산성을 뽑을 수 있다.

이 코스는 **"Claude Code가 내부적으로 어떻게 작동하는지"** 부터 시작해서 **"실전에서 어떻게 써먹는지"** 까지, 하나의 흐름으로 정리한 가이드다. 공식 문서를 전부 읽고, 소스 코드 구조를 분석하고, 실전 사례를 연구해서 만들었다.

## 누구를 위한 가이드인가?

```
이런 사람에게 딱 맞는 코스:

  [O] 개발 경험은 있다 (언어 무관)
  [O] Claude Code를 설치는 했는데, 제대로 활용하지 못하고 있다
  [O] "왜 이렇게 동작하지?"가 궁금한 호기심 많은 개발자
  [O] 팀에 Claude Code를 도입하려는 리드/시니어
  [O] CI/CD에 AI 에이전트를 통합하고 싶은 DevOps 엔지니어

이런 사람은 다른 자료를 추천:

  [X] 프로그래밍을 아예 모르는 비개발자
      -> 먼저 기본 프로그래밍을 배우세요
  [X] Claude API를 직접 호출해서 앱을 만들려는 사람
      -> Anthropic API 문서가 더 적합합니다
```

## 학습 로드맵

### Chapter 1: 에이전틱 루프 완전 해부 -- *예상 학습 시간: 60분*
> Claude Code의 심장을 열어본다.

사용자가 메시지를 보내면 내부에서 무슨 일이 벌어지는지, 6단계 순환 프로세스를 완전히 해부한다. 컨텍스트 조립, API 호출, 도구 디스패칭, 권한 파이프라인, 컴팩션까지. 이걸 이해하면 "왜 Claude가 이렇게 행동하는지"가 보이기 시작한다.

**핵심 토픽**: 에이전틱 루프, 도구 디스패칭, 컨텍스트 컴팩션, 서브 에이전트, SDK

### Chapter 2: 메모리, 컨텍스트 & 설정 시스템 -- *예상 학습 시간: 45분*
> Claude의 기억력을 설계한다.

CLAUDE.md의 4단계 계층 구조, @include 디렉티브, Rules 파일, settings.json, 환경 변수까지. Claude가 "어떤 정보를 기반으로 판단하는지"를 완전히 이해하고, 프로젝트에 맞게 설정하는 법을 배운다.

**핵심 토픽**: 4단계 메모리 계층, @include, Rules, settings.json, 환경 변수

### Chapter 3: Hooks, Skills, Permissions -- *예상 학습 시간: 75분*
> 자동화의 세계로 입장한다.

파일 저장 시 자동 포매팅, 위험한 명령어 자동 차단, 반복 워크플로우 원클릭 실행, MCP로 외부 서비스 연동. 수동으로 하던 일을 자동화하는 모든 메커니즘을 익힌다.

**핵심 토픽**: 20+ Hook 이벤트, 4가지 Hook 타입, Skills, MCP 서버, 5가지 권한 모드

### Chapter 4: CLI 명령어와 내장 도구 -- *예상 학습 시간: 50분*
> 실전 무기를 장전한다.

설치부터 인증, 30+ CLI 플래그, 16개 슬래시 명령어, 5종 내장 도구(Bash, File, Search, Web, Agent)의 모든 파라미터와 활용법. 키보드 단축키와 대화형/비대화형 모드까지 실전 레퍼런스로 정리한다.

**핵심 토픽**: CLI 플래그, 슬래시 명령어, 내장 도구 5종, 권한 모드, 비대화형 모드

### Chapter 5: 실전 활용과 베스트 프랙티스 -- *예상 학습 시간: 60분*
> 이론을 현실에 적용한다.

실전 프로젝트 워크플로우, CLAUDE.md 프로처럼 쓰는 법, 멀티 에이전트 실전, CI/CD 통합 패턴, 비용 관리, 흔한 실수와 해결법, 프로 팁 30+개. 이 챕터를 마치면 현업에서 바로 활용할 수 있다.

**핵심 토픽**: 실전 워크플로우, CLAUDE.md 작성법, CI/CD 통합, 비용 최적화, 프로 팁

## 시작할 준비가 되었나요?

Claude Code는 단순한 코드 생성기가 아니다. 파일을 읽고, 코드를 분석하고, 명령어를 실행하고, 테스트를 돌리고, 버그를 수정하는 **자율적인 개발 에이전트**다. 이 에이전트를 제대로 이해하고 제대로 다루면, 혼자서 팀 하나의 생산성을 낼 수 있다.
