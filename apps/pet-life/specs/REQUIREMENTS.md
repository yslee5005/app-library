# REQUIREMENTS.md — Pet Life

> Version: 1.0 | Last Updated: 2026-04-02

## 앱 이름: Pet Life (펫 라이프)
## 한줄 설명: 논문 기반 반려견 인생 대시보드 — 남은 시간, 건강 위험, 맞춤 가이드

---

## 핵심 기능

### 1. 온보딩 (첫 실행)
- Step 1: 강아지 이름 입력
- Step 2: 견종 선택 (검색 + 130종 목록, "기타" 옵션)
- Step 3: 생년월일 또는 대략적 나이 입력
- Step 4: 현재 체중 입력 (적정 범위 즉시 표시)
- Step 5: 중성화 여부

### 2. 메인 페이지 (가상 강아지)
- 가상 강아지 애니메이션 (시간대별 자동 변화)
  - 6-8시: 기상/기지개, 8-9시: 식사, 9-12시: 낮잠
  - 12-14시: 놀이, 14-17시: 오후 낮잠, 17-19시: 산책
  - 19-20시: 저녁식사, 20-22시: 휴식, 22-6시: 수면
- 시간대별 정보 메시지 (논문 기반)
- 오늘의 체크리스트 (산책, 식사, 약)
- 오늘의 인사이트 카드 (매일 다른 건강 정보)
- 남은 시간 프로그레스 바

### 3. 분석 페이지 (대시보드)
탭 구성: [시간] [건강] [발달] [비용]

#### 시간 탭
- 남은 시간 (일/년)
- 인생 진행률 (%)
- 남은 산책 횟수
- 남은 식사 횟수
- 사람 나이 환산

#### 건강 탭
- 견종별 유전병 위험 카드 (심각도 색상 코딩)
  - 각 카드: 질환명, 발생률(%), 심각도, 예방법
  - 탭 → 상세 페이지 (논문 출처, 나이별 위험도, 권장사항)
- 나이 기반 경고 (치주질환, 인지기능 저하, 비만)
- 체중 상태 (정상/과체중/비만 + 수명 영향)

#### 발달 탭
- 인생 타임라인 (탄생 → 사회화 → 성견 → 시니어 → 고령)
- 현재 위치 표시
- 나이별 마일스톤 설명

#### 비용 탭
- 남은 생애 예상 비용
- 비용 구성 (식비/의료비/기타)

### 4. 가이드 페이지
- 나이 맞춤 건강검진 체크리스트
- 예방접종 스케줄 (다음 접종 표시)
- 시니어 식이 가이드
- 운동 권장량
- 치매 예방 활동

### 5. 설정
- 다크모드
- 알림 설정 (건강검진 리마인더, 접종 알림)
- 강아지 정보 수정
- 언어 (한국어/영어)
