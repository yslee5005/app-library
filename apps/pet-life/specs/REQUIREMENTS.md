# REQUIREMENTS.md — Pet Life

> Version: 2.0 | Last Updated: 2026-04-02

## 앱 이름: Pet Life (펫 라이프)
## 한줄 설명: 말 못하는 반려견의 상태를 대변하는 앱 — 인생 여정, 건강 인사이트, 데일리 루틴
## 컨셉: "이 앱은 반려견의 목소리가 되어줍니다"

---

## 핵심 기능

### 1. 온보딩 (첫 실행)
- Step 1: 강아지 이름 입력
- Step 2: 견종 선택 (검색 + 130종 목록, "기타" 옵션)
- Step 3: 생년월일 또는 대략적 나이 입력
- Step 4: 현재 체중 입력 (적정 범위 즉시 표시)
- Step 5: 데일리 루틴 등록 (커스텀 가능)
  - 기본 추천: 아침산책, 저녁산책, 아침식사, 저녁식사
  - 선택: 양치질, 약 복용, 간식, 놀이시간
  - [+ 직접 추가] 가능
  - 견종 기반 추천 표시 ("골든 리트리버는 하루 60-120분 산책 권장")

### 2. 메인 홈
- 다크 프리미엄 배경 (#1C1C2E)
- 상단: 이름 + 스트릭 뱃지
- 히어로: Lottie/이미지 강아지 캐릭터 + 말풍선 (상태 반영)
  - 강아지가 앱의 상태를 "말"함 — 반려견 대변 컨셉
  - 상태별 메시지: 행복/산책요청/배고픔/슬픔/경고
- 인생 여정 미니 타임라인 바:
  - 탄생●━━━●━━━●━━🐕━●━━○━━○
  - 과거=앰버, 미래=회색, 현재=강아지 아이콘+글로우
  - "58% 경과" 표시
- 오늘의 루틴: 수평 스와이프 카드
  - 각 카드: 아이콘 + 루틴명 + 개별 스트릭 🔥 + 완료 버튼
  - 탭하면 완료 → 강아지 반응 애니메이션
- 인사이트 pill: 매일 다른 건강 정보 (논문 기반)
- 바텀 내비: 게이미피케이션 스타일 (XP바 + Lv표시)

### 3. 강아지 상태 시스템 (반려견 대변)
강아지 상태 = 유저 행동의 거울

#### 루틴 기반:
- 산책 완료 → "산책 다녀왔어요! 😊"
- 산책 1일 미완 → "산책 가고 싶어요... 🥺"
- 산책 3일 미완 → "3일째 산책을 못 갔어요. 지루해해요 😔" + 치매 경고
- 산책 7일 미완 → "많이 답답해해요. 산책이 필요해요 😢"
- 식사 완료 → "맛있게 먹었어요! 🍖"
- 양치 5일 미완 → "이가 걱정돼요. 80-90% 치주질환" + 출처
- 전부 완료 → "완벽한 하루! 오늘도 최고예요! 🎉"

#### 건강 기반 (견종+나이 자동):
- 7세 골든 시니어 → "시니어에 진입했어요. 연 2회 건강검진 추천"
- 암 위험 → "정기 검진이 중요해요. 골든의 60%가 암으로..."
- 과체중 → "체중이 적정보다 높아요. 수명이 2.5년..."

### 4. 스트릭 시스템
- 각 루틴별 독립 스트릭 (산책 🔥14, 식사 🔥30, 양치 🔥3)
- "완벽한 하루" 스트릭 (전체 루틴 완료 시)
- 스트릭 프리즈: 월 2회 무료
- 스트릭 끊기면 → 강아지 슬픈 상태 + 알림

### 5. 분석 페이지 (📊 정량 데이터 — 주기적 확인)

#### 시간 탭
- 남은 시간: 큰 숫자 + 원형 프로그레스 (1,898일)
- 남은 산책/식사 횟수
- 사람 나이 환산
- 인생 여정 타임라인 풀 뷰 (수직 스크롤)
  - 과거 (앰버): 탄생 → 사회화 → 성견 → 시니어
  - 현재 (글로우): "7세 · 54세 · 1,898일"
  - 미래 (회색+경고): 고관절 → 고령기 → 평균수명
- [공유하기] 버튼

#### 건강 탭
- 견종별 유전병 카드 (위험도 색상: 빨강/주황/노랑)
- 나이별 위험도 (치매 14% → 28%)
- 체중 상태 (적정/과체중 + 수명 영향)
- 탭 → 상세 페이지 (논문 출처 + 예방법)

#### 예방 탭 (구 가이드 → 분석으로 통합)
- 나이 맞춤 건강검진 체크리스트
- 예방접종 스케줄 (다음 접종 표시)
- 시니어 식이 권장
- 운동량 권장 (견종 기반)
- 치매 예방 활동

### 6. 기록 페이지 (📝 내 활동 — 성취감)

#### 달력 뷰
- 히트맵 캘린더 (완료한 날 앰버 색칠, 미완 회색, 미래 빈칸)
- 탭하면 해당 날 루틴 상세

#### 스트릭 현황
- 각 루틴별 현재 스트릭 + 최장 기록
- 🦴 산책 🔥14일 (최장: 30일)
- 🍖 식사 🔥30일 (최장: 45일)
- ⭐ 완벽한 하루 🔥5일

#### 월간 통계
- 이번 달 루틴 달성률 바 차트
- 총 산책 횟수/시간
- 전월 대비 변화

#### 📸 사진 타임라인 (v2)
- "오늘의 한 장" 모음 → 추억 타임라인

#### [월간 리포트 공유] 버튼

### 7. 설정
- 루틴 수정/추가/삭제
- 알림 시간 설정
- 강아지 정보 수정
- 다크모드 (기본 다크)
- 언어 (한국어/영어)
